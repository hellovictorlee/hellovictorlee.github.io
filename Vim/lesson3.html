Normal Mode & Command Line Mode

It's really worth learning more things of Normal Mode

It's time to know more about Command Line Mode

Find & Search

Normal Mode

  • find: f, t, F, T
  • keep find: ;, ,

Normal Mode

  • search word with cursor: #, *
  • next: n
  • previous: N

Command Line Mode

  • Search: /, ?
  • next: n
  • previous: N

Repeat

repeat: .

When to use?

  • correspond command: change, delete, replace
  • correspond command: c, d, x, r, R

delete one char where the cursor is: x

repeat only works when the command is used to change the content

Undo & Redo

undo: u

redo: Ctrl + r

Replace

replace: r, R

r only change on char

escape R: ESC

Moving around brace

jump between left and right brace: %

cursor in html tag would jump between tag

More movement

Move screen up one line: Ctrl-y

Move screen down line: Ctrl-e

Move cursor & screen up a half page: Ctrl-u

Move cursor & screen down a half page: Ctrl-d

Move cursor up one page & cursor to the last line: Ctrl-b

Move cursor down one page & cursor to the first line: Ctrl-f

Move current line to the middle of the screen: z + z

Move current line to the top of the screen: z + t

Move current line to the button of the screen: z + b

Windows manager

Rotate the windows: Ctrl-w + r

Swap the current window with the next one: Ctrl-w + x

Move current window to the first left: Ctrl-w + Shift-h

Move current window to the first right: Ctrl-w + Shift-l

Move current window to the top: Ctrl-w + Shift-k

Move current window to the button: Ctrl-w + Shift-j

Marks

make a mark: m + English letter

ex: ma, mb, mc,...

move cursor to the mark: ' + English letter

move cursor to the mark: ` + English letter

Go with marks

Integrate with delete, change, & yank
