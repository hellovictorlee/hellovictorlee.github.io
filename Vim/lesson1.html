INSERT MODE

typing...

NORMAL MODE

moving your cursor

  • DON'T use arrow key
  • use "h, j, k, l"

COMMAND-LINE MODE

doing something without typing and moving cursor

ex: save file, exit,...

  • save file: "w"
  • quit: "q"

HOW TO SWITCH MODE

Insert mode: "i"

Normal mode: ESC

Command-line mode: ":"
