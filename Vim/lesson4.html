Command Line Mode & Insert Mode

Switch to Command Line Mode, Insert Mode, Visual Mode must from Normal Mode

Review

  • to Command line mode: :
  • to Insert Mode: i
  • to Visual Mode: v

Basic Command Line Mode

Write: w

Quit: q

Force: !

Combine together. Ex: wq

Open file 1

Edit: e

Horizontal split: sp

Vertical split: vsp

Move panel: Prefix + Direction

  • Prefix: Ctrl + w
  • Direction: h, j, k, l

Open file 2

Tab edit: tabedit

  • Move forward: gt
  • Move backward: gT

Insert Mode

Insert mode is only for typing.

Insert: i, a, s, o, I, A, S, O

Autocomplete: Ctrl + n/p

  • Move downward: Ctrl + n
  • Move upward: Ctrl + p
